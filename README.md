# Forger 2 - Resource Gathering Game

A Forger-style resource gathering and building game built with Phaser 3.

## How to Play

### Controls
- **WASD / Arrow Keys** - Move your character
- **Space (Hold)** - Mine/harvest resources when near them

### Gameplay
1. **Gather Resources** - Walk near trees, rocks, ores and hold Space to harvest
2. **Craft Tools** - Open the Crafting tab to make better tools
3. **Build Structures** - Construct workbenches, forges, and more
4. **Upgrade** - Purchase upgrades to mine faster and carry more
5. **Explore Islands** - Unlock new islands with rarer resources

### Resources (by tier)
- **Tier 1**: Wood, Stone, Fiber, Berries
- **Tier 2**: Coal, Copper
- **Tier 3**: Iron
- **Tier 4**: Gold, Crystal
- **Tier 5**: Diamond, Mythril

### Tools
Craft better pickaxes and axes to harvest faster:
- Wooden → Stone → Copper → Iron → Gold → Diamond → Mythril

### Buildings
- **Workbench** - Craft basic items
- **Forge** - Smelt ores and craft metal tools
- **Storage** - Store extra resources
- **House** - Rest and restore energy
- **Portal** - Travel between islands
- **Mine** - Access underground resources

### Islands
Progress through 6 islands, each with different resources:
1. Starter Island
2. Copper Hills
3. Iron Mountains
4. Golden Valley
5. Crystal Caves
6. Mythril Peaks

## Development

```bash
npm install
npm run start
```

## Tech Stack
- Phaser 3 (game engine)
- Vite (build tool)
- Procedurally generated pixel art sprites
